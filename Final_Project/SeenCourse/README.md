# Seen user - Course prediction

## how to train?

 1. put all the dataset files in path **./hahow/data/\***
 2. run script **train.sh**
    ```
    bash train.sh
    ```

## how to test?
 1. put all the dataset files in path **./hahow/data/\***
 1. rename test data file as **./hahow/data/test_seen.csv**
 2. run script **run.sh**
    ```
    bash run.sh
    ```
